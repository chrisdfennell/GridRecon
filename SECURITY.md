# Security Policy

## Supported Versions

The latest released version of GridRecon receives fixes. Older tagged releases
are not maintained.

## Scope & data handling

GridRecon is a Garmin Connect IQ watch app. It requests the **Positioning** (GPS)
and **Sensor** (compass heading) permissions, used only to show your location and
point the return arrow. Saved marks are stored **on the device** via
`Application.Storage`. The app does **not** transmit any data off the watch.

## Reporting a Vulnerability

If you find a security or privacy issue, please report it privately rather than
opening a public issue:

- Use GitHub's [private vulnerability reporting](../../security/advisories/new), or
- Email **infantrykiller@gmail.com** with the details and steps to reproduce.

Please allow up to 7 days for an initial response. Thanks for helping keep the
project safe.
